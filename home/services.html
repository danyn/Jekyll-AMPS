SCSS

  • You can use scss
  • Inlines all css into the head
  • All page and component CSS is included conditionally per page and needs a bit of manual setup

Organization

  • You can keep scss files with the html page they style in the same folder
  • You can access scss variables from one location and override them
  • You keep images associated with the page in the same folder and use include to write an AMP img tag for it

Easy To Use

  • Everything is declarative only AMPS components are used
  • It Validates nicely
  • It is easy to modify and built using Minima as a starting point
